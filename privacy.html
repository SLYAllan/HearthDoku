Mentions légales & politique de confidentialité

1. Éditeur du site

Site édité à titre personnel et non commercial par Solary_Allan. Contact : @Solary_Allan sur X (Twitter).

2. Hébergement

Le site est hébergé par GitHub Pages (GitHub, Inc., 88 Colin P Kelly Jr St, San Francisco, CA 94107, USA), derrière le CDN Fastly.

3. Propriété intellectuelle

HearthDoku est un projet non officiel, sans affiliation avec Blizzard Entertainment. Les cartes, illustrations et marques Hearthstone sont la propriété exclusive de Blizzard Entertainment, Inc.

Le code source du site est publié sur GitHub.

4. Données personnelles & cookies

Aucune inscription n'est requise pour jouer. Le site ne collecte pas de données permettant de vous identifier personnellement (pas de compte, pas d'e-mail, pas de profil).

4.1. Stockage local (localStorage)

Le jeu conserve localement dans votre navigateur, et uniquement sur votre appareil :

  • votre préférence de langue (hearthdoku_lang) ;
  • votre choix de consentement aux cookies (hearthdoku_consent) ;
  • votre progression sur le puzzle quotidien.

Ces informations ne quittent jamais votre navigateur.

4.2. Mesure d'audience (Google Analytics via Google Tag Manager)

Si vous acceptez dans la bannière de consentement, Google Analytics est chargé pour mesurer l'audience du site (pages vues, fréquentation). Aucun outil de tracking n'est chargé tant que votre consentement n'est pas donné.

Vous pouvez à tout moment revenir sur votre choix via le bouton « Gérer mes cookies » dans le pied de page.

4.3. Vos droits (RGPD)

Conformément au RGPD, vous disposez d'un droit d'accès, de rectification, d'effacement et d'opposition concernant vos données. Comme aucune donnée identifiante n'est collectée côté serveur par ce site, ces droits s'exercent directement :

  • en refusant / retirant le consentement à la mesure d'audience ;
  • en effaçant le stockage local de votre navigateur pour ce site ;
  • auprès de Google pour les traitements opérés par Google Analytics (policies.google.com/privacy).

Vous pouvez également introduire une réclamation auprès de la CNIL.

5. Données de cartes Hearthstone

Les données des cartes sont fournies par l'API publique HearthstoneJSON. Les images de cartes éventuellement chargées proviennent également de ce service et ne transitent pas par nos serveurs.

6. Contact

Pour toute question relative à ces mentions : @Solary_Allan ou via une issue GitHub.
